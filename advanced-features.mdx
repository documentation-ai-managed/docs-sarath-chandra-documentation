## Collaboration Tools

<Steps>
  <Step title="Real-time Editing" icon="users">
    Multiple users can edit documents simultaneously.
  </Step>
  <Step title="Comments & Reviews" icon="message-square">
    Add comments and request reviews for changes.
  </Step>
  <Step title="Permissions" icon="shield">
    Control access levels for different team members.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ````javascript
  const collaboration = {
    enableRealTime: true,
    allowComments: true,
    permissions: {
      read: ['viewer'],
      write: ['editor'],
      admin: ['admin']
    }
  };
  
  docs.updateProjectSettings(projectId, collaboration);
  ````
  ````python
  collaboration = {
      'enable_real_time': True,
      'allow_comments': True,
      'permissions': {
          'read': ['viewer'],
          'write': ['editor'],
          'admin': ['admin']
      }
  }
  
  docs.update_project_settings(project_id, collaboration)
  ````
</CodeGroup>
